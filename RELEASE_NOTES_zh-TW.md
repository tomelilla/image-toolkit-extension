# 版本資訊 (Release Notes)

## [1.0.0] - 2025-12-09

### 新增功能 (Added)
- **多圖支援 (Multi-Image Support)**：支援一次上傳多張圖片，並可透過上方列表快速切換編輯。
- **智慧拼接 (Smart Stitching)**：
    - **自動統一尺寸**：拼接前自動調整圖片高度 (水平模式) 或寬度 (垂直模式)。
    - **智慧移除重疊**：新增「自動移除重疊」選項，自動偵測並削除長截圖中的重複區域。
    - **效能優化**：採用縮圖分析演算法，大幅降低記憶體佔用與處理時間。
- **圖片壓縮升級**：
    - 支援多種輸出格式：**JPEG** (預設)、**PNG**、**WebP**。
    - 可調整壓縮品質。
- **介面優化 (UI Improvements)**：
    - **分頁設計**：將工具整理為分頁 (壓縮、縮放、裁切、拼接)，操作更直覺。
    - **拖放上傳**：優化圖片拖放區域。
    - **輸出資訊**：顯示處理後的檔案大小與尺寸。
    - **時間戳記檔名**：下載的圖片檔名包含時間戳記，方便管理。

### 移除功能 (Removed)
- **移除背景**：應需求移除該功能。

### 修正 (Fixed)
- **拼接修正**：修正不同尺寸圖片拼接時的對齊問題。
- **效能修正**：解決高解析度圖片拼接時的記憶體不足問題。
