# 圖片工具包 Chrome 擴充功能

**作者：** GeminiCLI {gemini-1.5-pro-002}

此 Chrome 擴充功能在您的瀏覽器彈出視窗中提供了一套完整的圖片處理工具。專為效率與隱私設計，所有圖片均在「本地端」進行處理，無需將圖片上傳至任何伺服器。

## 主要功能

### 1. 多圖管理功能
- **列表視圖**：支援拖放或點擊上傳多張圖片。
- **快速切換**：透過縮圖列表，輕鬆在不同圖片間切換編輯。
- **批次準備**：上傳的所有圖片皆可直接用於拼接功能。

### 2. 智慧圖片拼接 (Smart Stitching)
將多張截圖或圖片合併為單一長圖。
- **拼接方向**：支援 **水平** 與 **垂直** 拼接。
- **自動統一尺寸**：自動調整圖片大小 (例如垂直拼接時統一寬度)，確保拼接邊緣整齊。
- **智慧移除重疊 (Smart Stitch)**：
    - 自動偵測並移除圖片之間的重疊區域。
    - 非常適合用於製作「長截圖」 (將多張滾動截圖接合)。
    - 優化演算法，確保高解析度圖片也能快速處理。

### 3. 圖片壓縮
在不犧牲品質的情況下減少檔案大小。
- **格式支援**：可輸出為 **JPEG**、**PNG** 或 **WebP**。
- **品質控制**：提供 0-100 的品質滑桿，讓您自由平衡檔案大小與畫質。

### 4. 圖片編輯工具
- **裁切**：提供自由裁切或預設比例 (正方形、16:9、圓形)。
- **縮放**：依百分比等比例縮放圖片。
- **分析**：查看圖片詳細資訊 (尺寸、格式)。

### 5. 隱私優先 & 本地運算
- 所有操作皆在您的 **瀏覽器內** 完成。
- **無資料上傳**：您的圖片永遠不會離開您的裝置，確保絕對隱私。

## 安裝說明

1.  **下載擴充功能**：
    *   下載此專案程式碼或 Clone 儲存庫。

2.  **開啟擴充功能管理頁面**：
    *   在 Chrome 網址列輸入 `chrome://extensions/` 並按下 Enter。

3.  **啟用開發者模式**：
    *   開啟右上角的「開發者模式 (Developer mode)」開關。

4.  **載入擴充功能**：
    *   點擊「載入未封裝項目 (Load unpacked)」。
    *   選擇本專案的資料夾。

## 使用方式

1.  **上傳**：點擊擴充功能圖示，將圖片拖入虛線區域。
2.  **編輯**：使用分頁切換不同工具 (壓縮、縮放、裁切、拼接)。
3.  **拼接**：若要拼接，請確保已上傳多張圖片。切換至 **拼接 (Stitch)** 分頁並點擊按鈕。若為長截圖，請勾選「自動移除重疊」。
4.  **下載**：處理完成後點擊「下載圖片」，檔案會自動加上時間戳記。
