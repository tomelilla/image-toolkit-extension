# 圖片工具包 Chrome 擴充功能

**作者：GeminiCLI {gemini-1.5-flash-001}**

此 Chrome 擴充功能在您的瀏覽器彈出視窗中提供了一套完整的圖片處理工具。它允許使用者對上傳的圖片執行各種操作，包括分析、縮放、壓縮、裁切和拼接。

## 功能

*   **圖片分析**：查看圖片的詳細資訊（檔案名稱、副檔名、寬度、高度）。
*   **等比例縮放**：依指定百分比調整圖片大小，同時保持長寬比。
*   **圖片壓縮**：透過調整品質來減少圖片檔案大小（支援 JPEG/PNG）。
*   **圖片裁切**：根據 X、Y 座標以及指定的寬度和高度來裁切圖片的特定區域。
*   **圖片去背（預留位置）**：此功能由於其複雜性而預留。在實際生產環境中，這通常需要進階演算法或整合外部 API。
*   **圖片拼接**：將多張圖片水平或垂直合併為一張圖片。

## 安裝

要安裝和使用圖片工具包擴充功能：

1.  **下載擴充功能**：
    *   複製此儲存庫或下載 `image-toolkit-extension` 資料夾。

2.  **開啟 Chrome 擴充功能頁面**：
    *   開啟您的 Chrome 瀏覽器。
    *   在網址列輸入 `chrome://extensions/` 並按下 Enter。

3.  **啟用開發者模式**：
    *   在擴充功能頁面上，切換開啟「開發者模式」開關，通常位於右上方。

4.  **載入未封裝擴充功能**：
    *   點擊「載入未封裝項目」按鈕。
    *   導航至您下載/複製 `image-toolkit-extension` 資料夾的位置。
    *   選擇 `image-toolkit-extension` 資料夾並點擊「選取」。

5.  **釘選擴充功能（可選）**：
    *   載入後，您的 Chrome 工具列中會出現一個拼圖圖示。點擊它。
    *   在列表中找到「圖片工具包」並點擊旁邊的釘選圖示，以便在工具列中輕鬆存取。

## 使用方式

1.  **開啟擴充功能**：點擊 Chrome 工具列中的「圖片工具包」圖示。
2.  **上傳圖片**：使用「上傳圖片：」輸入欄位從您的電腦中選擇一張圖片。原始圖片將會顯示。
3.  **使用工具**：
    *   **分析**：點擊「分析」以查看其尺寸、檔案名稱和副檔名。
    *   **縮放**：輸入百分比並點擊「縮放」以等比例調整圖片大小。
    *   **壓縮**：輸入品質值 (0-100) 並點擊「壓縮」以減少檔案大小。
    *   **裁切**：輸入 X、Y 座標以及裁切區域的所需寬度和高度，然後點擊「裁切」。
    *   **去背**：這是一個預留位置按鈕。完整的實作將需要進階圖片處理。
    *   **拼接**：使用「上傳拼接圖片：」輸入欄位選擇多張圖片，選擇方向（水平/垂直），然後點擊「拼接」。
4.  **下載輸出**：處理後，輸出圖片將會顯示。點擊「下載圖片」將其儲存到您的電腦。

## 本地化 (i18n)

此擴充功能支援多種語言。Chrome 會自動使用與您瀏覽器 UI 語言相符的語言（如果可用）。支援的語言包括：

*   英文（預設）
*   繁體中文
*   簡體中文
*   日本語

要更改擴充功能的語言，您需要更改 Chrome 瀏覽器的顯示語言設定。
